LAYOUTS criativos para aulas inovadoras

É um guia para professores que desejam repensar o espaço da sala de aula e criar estratégias de aprendizagem mais envolventes e memoráveis para os estudantes. Através da metodologia Blueprints, o livro propõe a alteração do layout da sala de aula para criar condições para a aplicação de estratégias de aprendizagem inovadoras. O objetivo é estimular práticas de inovação, inspiração e cocriação entre professor e estudante, para facilitar o desenvolvimento de novas metodologias para sala de aula. Com este livro, os professores são convidados a abraçar o papel de arquitetos da educação, desenhando os planos para um ambiente de aprendizagem que vai além das quatro paredes e das fileiras de carteiras. É uma leitura recomendada para aqueles que desejam repensar o espaço educacional e criar experiências de aprendizagem mais envolventes e inspiradoras.

‍Autoras: Ana Valéria S. A. Reis, Karina Nones Tomelin e Thuinie Daros
